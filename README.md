### 본격 효자 만들기 프로젝트!! 

#### 대상

- 주요 대상 : 부모님과의 추억을 기록하고 싶은 자녀
- 가족들도 공유 가능

#### 네비게이션

- 앨범 / 전체 / 업로드 / 미션 / 설정 

#### 알림 

- 생일인 사람 있는 가족 알림 추가 

#### 호칭 테이블

- 복잡하게 할지 / 간단하게 할지 

#### 미션

- 등급 (티어)
- 미션, 포인트 (최대한 사진으로 인식)
- 포즈도 가능하면?
- 공유하면 추가 점수
- 기간 챌린지 (20개)
- 미션은 템플릿 정해놓고 그 중에서 랜덤
- 잔디밭 관리처럼 보여주면 괜찮을 듯 

#### 웹

- 사진 보기만 (앨범, 전체)
- 회원가입 x, 로그인 o
- 간단한 개인정보 (이름, 계정)
