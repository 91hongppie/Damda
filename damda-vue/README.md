# damda-vue

### 설치

| Node.js 설치 - https://nodejs.org/ko/download/ 에 들어가서 OS 환경에 맞는 버전 설치

```bash
# 필요 라이브러리 설치
npm install
```



### 프로젝트 실행

```bash
npm run serve
```
